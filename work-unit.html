Peeper's Dallas

Overview

The ILOs are game-like animations in the courses.

Goals

  • Update the assets of the animations.
  • Retain teaching function of the animations.
  • Introduce active decision-making where there is only a passive experience.
  • Must especially consider the experience on mobile devices.
  • Built with open technologies, such as HTML, CSS, JS

Challenges

  • I've never designed an animated interface before this.
  • Leading a project is a new experience for me.
  • When starting the project, I was a JavaScript noob. Like, bigtime. (I've worked with experts, both contracted and internal.)
