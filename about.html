I'm a guy from a small-town in Louisiana living in Dallas, TX with my wife and dog. We were scared to death to drive on the interstates when we first moved here. It took us forever to get anywhere. Now I'm a big fan of public transit and only a little bit scared of the interstates.

I'm working at Kalkomey doing design, front-end development, and animation with a lot of smart people building experiences to teach people how to be safe while having fun.

Take a look at my work, or if you're looking for more talented folks check out my friends and recommendations.

Contact

The best way to contact me is through Twitter, email, or we can just grab a beer or something. I'm happy to talk about whatever.

Colophon

This site is built with Jekyll, a static site generator. The typefaces used are FF Meta and FF Meta Serif, served up by Typekit. The grid is built with Gridset, one of the best web tools for building a responsive web I've ever used. View my grid system, and to view the overlay, add ?gridset=show to the URL (on any page).

Source code for this site can be found on Github.
